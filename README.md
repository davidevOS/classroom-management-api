# Classroom Management API
API for classrooms' management

# RESTful Endpoints
* List of teachers
    * GET /teachers
* Teacher info by ID
    * GET /teachers/:id
* List of Students    
    * GET /students
* Student Info by ID
    * GET /students/:id
* List of grades
    * GET /grades
* Grades by student ID    
    * GET /grades/:id
* Update grades by Student ID
    * PUT /grades/:id
