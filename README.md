# Classroom Management API
API for classrooms' management

# RESTful Requests
* Get List of teachers
`GET /teachers`
* Get Teacher info by ID
`GET /teachers/:id`
* Get List of Students    
`GET /students`
* Get Student Info by ID
`GET /students/:id`
* Get List of grades
`GET /grades`
* Get Grades by student ID    
`GET /grades/:id`
* Update grades by Student ID
`PUT /grades/:id`
